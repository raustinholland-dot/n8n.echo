## Cybersecurity Program Leadership & Transformation
- vCISO / Program Leadership: leaders to design/run standards-based program; use when client lacks senior leadership; outputs: roadmap, governance, exec reporting
- Strategic & Tactical Planning: framework-aligned program planning; use when sequencing initiatives; outputs: prioritized plan, milestones, exec-ready status
- Training & Workforce Enablement: workforce enablement; use for skill gaps/turnover; outputs: training plan + reinforcement guidance

---

## Risk Analysis & Risk Management
- HIPAA Risk Analysis (OCR-aligned): defensible baseline; outputs: risk analysis report, risk register, remediation plan
- Risk Mgmt Planning & Execution Support: drive response decisions; outputs: action plans, tracking, status reporting
- Maturity Assessment & Trending: measure progress/justify investment; outputs: baseline, benchmarks, trend reporting

---

## Technical Testing & Engineering
- Vulnerability Scanning: identify/prioritize vulns; outputs: findings + remediation recommendations
- Penetration Testing: simulated attacks; outputs: exploit paths + prioritized fixes
- Web/Mobile App Pen Test: app weaknesses; outputs: app sec report + fixes
- Phishing/Social Engineering: human-factor risk; outputs: campaign results + training actions
- Security Engineering & Remediation Support: implement/improve controls; outputs: guidance, configs, project artifacts
- Cloud Security (Assessment & Design): secure-by-design cloud; outputs: architecture assessment + hardening plan
- Network Assessment & Design: perimeter/segmentation improvements; outputs: assessment + design recommendations
- SCVA: control effectiveness vs realistic scenarios; outputs: results + recommendations
- Medical Device & IoT Security: device discovery/vulns; outputs: inventory insights + program recommendations

## Resiliency & Incident Readiness
- BIA: critical processes + RTO/RPO; outputs: BIA report + recovery requirements

- Ransomware Readiness Review: focused preparedness; outputs: gap list + actions
- IR Plan/Playbook Development: standardize response; outputs: plans, playbooks, escalation paths
- IR Tabletop Exercise: test decision-making; outputs: lessons learned + remediation actions
